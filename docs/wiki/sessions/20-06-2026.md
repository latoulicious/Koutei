# Decisions / Deviations from LazyScan

- **No `database.md`** — Koutei is stateless, holds no DB. Replaced by `data.md`
  (request payload + localStorage + embedded static seed).
- **Created root `AGENTS.md`** — the wiki references it as authoritative; without it
  the references dangle. Minimal, adapted from LazyScan.
- **Deferred `postman/`** — no API code exists yet; a stub collection would lie.
  Tracked as a follow-up.
- **Backend documented as Go** — the blueprint commits to Go + goroutines; Rust is an
  aside. A language ADR goes in `decisions/` if Rust is ever chosen.
- Content is design-intent placeholders, explicitly marked — not speculative
  invention. Code stays the source of truth.
